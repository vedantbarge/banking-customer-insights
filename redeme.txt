🏦 Banking Customer Insights Dashboard

A complete end-to-end data analysis and dashboard project using MySQL, Python, and Power BI.

📊 Project Objective

To analyze banking customer data and generate actionable insights into:

Customer demographics
Loan and deposit product usage
Risk-weighting and tenure patterns
🔧 Tools Used
MySQL
Python (Pandas, NumPy, Seaborn, Matplotlib)
Power BI
GitHub (version control)
🧾 What’s Inside
📂 Python Analysis (banking_analysis.py)
📈 Power BI Dashboard (Loan, Deposit, Summary)
📘 Final Report (PDF with insights and findings)
📌 Key Highlights
Performed data cleaning and transformation using Python
Conducted exploratory data analysis (EDA) to identify trends
Built interactive dashboards in Power BI
Generated business insights for better decision-making
